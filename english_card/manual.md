English Word ユーザーマニュアル
English Wordは、ブラウザ上で動作する軽量かつ高機能な英単語学習アプリケーションです。フラッシュカード、タイピング、リスニングの3つのモードを通じて、効率的に英単語を学習できます。
主要機能
本アプリは、学習目的に合わせて3つのモードを提供します。
1. Cards (フラッシュカードモード)
単語のスペルと音声から、意味を推測するモードです。
表面: 英単語、発音記号、品詞が表示されます。
裏面: タップ（クリック）すると、日本語の意味と例文が表示されます。
判定: 「わかる」「わからない」のボタンで自己評価を行います。「わからない」を選択した単語は、セッション終了後にまとめて復習できます。
2. Typing (タイピングモード)
英単語のスペルを正確に入力するモードです。
画面に表示される日本語の意味と発音をヒントに、英単語をタイピングします。
正確に入力して Enter を押すことで次の単語に進みます。
一度でも入力を間違えたり、Skip を選択した単語は不正解として記録され、後で復習できます。
3. Reading (リスニングモード)
単語と例文の音声を連続して再生し、リスニング学習を行うモードです。
英単語 → 日本語の意味 → 英文（コンテキスト） の順で自動的に音声が再生されます。
画面を見なくても学習できるため、バックグラウンドでのリスニング学習に最適です。
再生・一時停止の切り替えが可能です。
スタート画面（設定）
学習を開始する前に、以下の設定をカスタマイズできます。
Questions (出題数): 1セッションあたりの問題数を 50、100、ALL (すべて) から選択します。
Order (出題順):
ID Order: 単語データのID順に出題します。「Start At」で開始するIDを指定できます。
Random: ランダムな順番で出題します。
Auto Audio (自動音声): ON にすると、CardsモードおよびTypingモードで問題が表示された瞬間に英単語の音声が自動再生されます。
Master Volume (音量): アプリ全体の読み上げ音量を調整します。
単語データのインポート (Import JSON)
標準で収録されている単語以外にも、独自の単語データを読み込んで学習できます。スタート画面の Import JSON ボタンをクリックしてデータを読み込みます。
インポート方法
Load File: パソコン内にある .json ファイルを選択して読み込みます。
Paste: クリップボードにコピーしたJSONデータを貼り付けます。
データが正常に認識されると「JSON Valid」と表示され、Save ボタンで適用されます。
JSONデータのフォーマット
インポートするデータは、以下のプロパティを持つJSON配列形式である必要があります。
[
 {
   "id": 1,
   "word": "prescriptive",
   "pronunciation": "/prɪˈskrɪptɪv/",
   "pos": "Adjective",
   "meaning": "規範的な、指示する",
   "wrong": "自由放任の",
   "context": "The manual offers prescriptive rules for grammar and punctuation.",
   "source": "Chicago Manual of Style"
 }
]
必須フィールド:
id (数値): 単語の固有ID
word (文字列): 英単語
pos (文字列): 品詞（Noun, Verb, Adjective など）
meaning (文字列): 日本語の意味
オプションフィールド:
pronunciation (文字列): 発音記号
context (文字列): 例文や文脈
wrong (文字列): 誤答の選択肢（※将来的な機能拡張用）
source (文字列): 出典やカテゴリ
復習機能
CardsモードおよびTypingモードのセッション終了後（Result画面）において、不正解だった単語がある場合、Review Unknown ボタンが表示されます。これをクリックすると、間違えた単語のみを対象とした復習セッションを即座に開始できます。
動作環境
本アプリケーションは、外部サーバーを必要とせず、単一のHTMLファイルとしてブラウザ上で完全に動作します。
推奨ブラウザ: Google Chrome, Microsoft Edge, Safari (最新版)
音声合成: ブラウザ標準の Web Speech API を使用します（デバイスの音声設定に依存します）。
バックグラウンド再生: MediaSession API に対応しており、スマートフォンのロック画面などからReadingモードの再生・一時停止・スキップ操作が可能です。
